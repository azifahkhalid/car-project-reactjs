# Car Web ReactJS
Frontend web using ReactJS dependencies 
- ReactJS
- CSS
(only frontend)


## Project setup
```
yarn
```

## Run in development mode
```
yarn start
```

### Launches the test runner
```
yarn test
```

### Compiles and minifies for production
```
yarn build
```
